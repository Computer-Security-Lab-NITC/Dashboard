Why Participate

Open Internet CTF
This is an open internet event. Participants may search, read, learn, and use online references during the competition.
Learn the Tools Beforehand
We provide a dedicated Tools page to help you prepare before the CTF. You are expected to install these on your systems.
Beginner Friendly
New to CTFs? No problem. This event is designed to be approachable while still being fun and competitive.

Instructions

Before the event, please make sure you complete the following:

  1. Join the WhatsApp group using the link shared to the registered mobile number entered in the Google Form.
  2. Go through the Tools page and learn the tools. You are expected to install these on your systems beforehand.
  3. Bring your own system for the CTF and make sure your laptop has a LAN port for connection.
  4. Always check the Updates page for the latest announcements.
